# Optimization Results

This folder contains sample output from the advertising schedule optimization system.

## Files

- `12月广告最终安排_with-mab-hol.xlsx` — Optimized December schedule **with** Multi-Armed Bandit exploration and holiday weighting
- `12月广告最终安排_mab-without-hol.xlsx` — Optimized December schedule **with** MAB but **without** holiday weighting

## Format

Each Excel file contains two sheets:
- **计划明细** (Plan Details) — Day-by-day placement schedule with per-row metrics
- **汇总结果** (Summary) — Per-activity aggregated statistics

> Note: The raw input data files are excluded from the repository due to size and sensitivity.
